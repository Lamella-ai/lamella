{# Minimized-jobs dock — pure client-side. Each pill represents a modal the user explicitly minimized via its "—" button. Clicking the pill removes the entry from sessionStorage and fetches the modal partial for that job, restoring the same overlay on the current page. Survives navigation within the tab (sessionStorage) but not across tabs or tab close — by design. The previous server-polled /jobs/active/dock dock was retired because it showed every active job (not just minimized ones) and clicking it sent the user to a full-page detail view that looked nothing like the original hovering modal. #}
