Step 5: Backfill historical payments?

After this wizard commits the loan + anchor, you can optionally run the historical-payment backfill flow to enter the months between funded date and anchor date. This is useful for tax reporting (interest paid YTD), debt-paydown reporting (principal paid since funding), and amortization-drift detection.

Skipping is fine — the loan's coverage panel will show "no payments before anchor" and start tracking from there. You can run backfill later from the loan detail page if you change your mind.
