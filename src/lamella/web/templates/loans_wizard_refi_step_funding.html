Step 5: New loan funding

In a typical refi, the new loan's principal flows directly to the old loan's payoff — so the offset account defaults to the old loan's liability path. The funding transaction posts -principal on the new liability and +principal on the old liability, zeroing out the old loan in one move.

For a cash-out refi, set the offset to a checking account and record the cash side via /record-payment after the wizard.
