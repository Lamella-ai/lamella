Display name

What gets shown in account pickers, the dashboard, and reports — anywhere the raw Beancount path would be ugly. Example: a path like Liabilities:Acme:Card:4821 reads better as “Acme Visa · 4821”.

Leave blank to fall back to the account path or the auto-generated label.

Kind

Used by the AI classifier to decide what kinds of transactions can land here, and by the cards / loans / vehicles UIs to know which account types to surface.

Common values: checking, savings, credit_card, loan, cash, investment, asset_book_value.

Entity

Determines which Schedule (C / E / A) this account’s activity rolls up under at tax time. Set it on every account you actively use; leave blank only on system accounts (clearing, opening balances).

SimpleFIN account ID

Paste the SimpleFIN account_id that maps to this ledger account so the periodic pull lands transactions here.

One ledger account ↔ one SimpleFIN account. If a single ledger account aggregates several real bank accounts, leave this blank and split via overrides.
