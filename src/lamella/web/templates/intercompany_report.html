How to record a settlement

When the owing entity pays the paying entity back via a cash transfer, record a clearing transaction in manual_transactions.bean — four legs for the settlement itself plus the actual money movement. Example for an Acme ← Example LLC 100 USD settlement: 

2026-05-01 * "Intercompany settlement: ExampleLLC → Acme" #lamella-intercompany-settlement
  lamella-intercompany-settlement-of: "<first-override-hash>"
  Liabilities:ExampleLLC:DueTo:Acme       100.00 USD    ; clears payable
  Assets:Acme:DueFrom:ExampleLLC         -100.00 USD    ; clears receivable

2026-05-01 * "Transfer ExampleLLC → Acme for intercompany settlement"
  Assets:ExampleLLC:Checking  -100.00 USD
  Assets:Acme:Checking         100.00 USD
