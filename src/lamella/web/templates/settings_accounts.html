Kind

Drives which UI surfaces this account: checking + savings show on dashboards; credit_card in the cards UI; loan in the loans UI; asset_book_value for vehicles + properties.

Saves you from setting up the obvious siblings by hand. For a checking account that means an opening-balance equity row; for a credit card it means matching interest + fee expense accounts; for a savings account it means an interest-income account.

Uncheck only if you intend to wire these up manually for a non-standard layout.
